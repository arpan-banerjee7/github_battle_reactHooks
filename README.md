
# GitHub Project refactored to use React Hooks

### Project

This project is a "Github Battle" app. You'll be able to see the most popular repos for a variety of languages as well as battle two Github users to see who has the better profile.

You can view the final project at __[https://affectionate-kirch-e5d424.netlify.app/](https://affectionate-kirch-e5d424.netlify.app/)__
